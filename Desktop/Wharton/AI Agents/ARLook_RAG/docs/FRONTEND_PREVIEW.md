## 📱 界面特性

### 左侧地图区域（占满剩余空间）
- **Google Maps** 全屏显示
- **初始状态**: 显示所有5个房源标记点
- **标记点样式**: 
  - 蓝色圆形标记 (#0ea5e9)
  - 白色边框
  - 价格标签（如 "$1800"）
- **交互**: 点击标记点显示房源详细信息

### 右侧聊天栏（固定380px宽度）
- **顶部**: 3个预设提示词按钮
  - "Wharton附近学生公寓"
  - "带室内洗烘房源"
  - "预算$1500-2000"
  
- **中部**: 消息列表
  - 用户消息: 蓝色背景，右对齐
  - AI消息: 灰色背景，左对齐
  - 自动滚动到最新消息
  
- **底部**: 输入框和发送按钮
  - 支持回车键发送
  - 加载时禁用

## 🎨 颜色方案

- **主色调**: 蓝色 (#0ea5e9)
- **用户消息**: 蓝色背景 (#2563eb)
- **AI消息**: 灰色背景 (#f3f4f6)
- **按钮**: 蓝色，悬停时变深蓝色
- **输入框**: 灰色边框，聚焦时蓝色边框

## 🚀 启动预览

要查看实际界面，请运行：

```bash
npm install
npm run dev
```

然后在浏览器中打开: http://localhost:3000
